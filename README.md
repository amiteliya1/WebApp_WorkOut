# Workout Tracker (מעקב אימונים)

פרויקט זה הוא אפליקציית Web שנבנתה באמצעות **React + Vite**. האתר מאפשר למשתמשים לנהל שגרת אימונים, ליצור תוכניות אימון מותאמות אישית, ולצפות במידע על תרגילים מתוך מאגר נתונים חיצוני.

האתר מעוצב בסגנון מודרני (Dark Mode) עם דגש על חוויית משתמש (UX/UI) ורספונסיביות מלאה (Mobile-First).

---

## 📋 מיפוי דרישות המטלה (3 העמודים הנדרשים)

הפרויקט כולל את שלושת העמודים הנדרשים על פי ההנחיות:

### 1. עמוד תוכן (Home / Content Page)
*   **קובץ:** `src/components/HomePage.jsx`
*   **תיאור:** עמוד הבית המציג את תוכנית האימונים השבועית בצורה ויזואלית (כרטיסיות). המשתמש יכול לראות את המיקוד של כל יום (למשל: "חזה וכתפיים"), וללחוץ על יום כדי לראות את פירוט התרגילים שנשמרו עבורו בחלונית (Modal).

### 2. עמוד טופס (Form Page)
*   **קובץ:** `src/components/WorkoutLogForm.jsx`
*   **תיאור:** עמוד המאפשר למשתמש ליצור ולהוסיף תרגילים לתוכנית האימונים.
*   **תכונות:**
    *   טופס עם שדות קלט (שם תרגיל, משקל, סטים, חזרות, יום בשבוע).
    *   ולידציה (Validation) לכל השדות לפני שמירה.
    *   שמירה ב-LocalStorage (כדי שהנתונים יישמרו גם לאחר רענון).
    *   הצגת משוב למשתמש (Modal הצלחה) לאחר שמירה.

### 3. עמוד API (API Page)
*   **קובץ:** `src/components/ExerciseApiPage.jsx`
*   **תיאור:** עמוד המתממשק ל-API חיצוני ומציג רשימה של תרגילים וקטגוריות.
*   **תכונות:**
    *   שליפת נתונים (Fetch/Axios) משרת מרוחק.
    *   הצגת הנתונים למשתמש בצורה מסודרת.

## 🛠️ התקנה והרצה (Installation & Setup)

כדי להריץ את הפרויקט במחשב המקומי, יש לבצע את השלבים הבאים:

1.  **התקנת הספריות (Dependencies):**
    פתח את הטרמינל בתיקיית הפרויקט והרץ:
    ```bash
    npm install
    ```

2.  **הרצת שרת הפיתוח (Development Server):**
    לאחר ההתקנה, הרץ את הפקודה:
    ```bash
    npm run dev
    ```

3.  **פתיחה בדפדפן:**
    לחץ על הקישור שמופיע בטרמינל (בדרך כלל `http://localhost:5173`) כדי לראות את האתר.
